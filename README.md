# ether-ai-deepseek

Adapter de Ether para DeepSeek implementado con `java.net.http.HttpClient` y `ether-json`, consumiendo la API HTTP directamente y mapeando las respuestas a los contratos definidos en `ether-ai-core`.

## Maven Dependency

```xml
<dependency>
    <groupId>dev.rafex.ether.ai</groupId>
    <artifactId>ether-ai-deepseek</artifactId>
    <version>8.0.0-SNAPSHOT</version>
</dependency>
```
